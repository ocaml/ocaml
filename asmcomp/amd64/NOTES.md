# Supported platforms

Intel and AMD x86 processors in 64-bit mode, a.k.a `x86_64`.

Floating-point architecture: SSE2, supported by all x86_64 processors.

Operating systems: Linux, BSD, MacOS X, MS Windows.

Debian architecture name: `amd64`
